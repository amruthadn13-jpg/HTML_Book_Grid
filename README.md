# HTML_Book_Grid

# 📚 Library Book Grid (HTML & CSS)

A simple **Library Book Grid webpage** created using **HTML and CSS**.
This mini project displays books in a **grid-style layout** using styled cards.

The project is part of my **web development learning journey**, where I am practicing how to structure webpages and style them using CSS.

---

## 🌐 Project Overview

This webpage displays a collection of books using **card-style boxes**.
Each book is shown with:

* Book Title
* Author Name

The layout uses **CSS styling** to make the book list look organized and visually clear.

---

## 🧠 Concepts Practiced

While building this project, I practiced:

* HTML page structure
* Using `<div>` containers
* CSS styling
* Background colors
* Borders and rounded corners
* Spacing using `margin` and `padding`
* Inline-block layout for grid-like display
* Header styling

---

## 🎨 Features

* Styled **header section**
* **Book cards** displayed in grid format
* Clean and simple **CSS design**
* Responsive-style layout using `display: inline-block`
